# To start application (Follow the below steps)
### Frontend Part
```bash
  git clone https://github.com/AMANKUMAR1020/tutedude-frontend
```
```bash
  cd tutedude-frontend
```
```bash
  npm i
```
```bash
  npm start
```

### Backend Part
```bash
  git clone https://github.com/AMANKUMAR1020/tutedudeServer
```
```bash
  cd tutedudeServer
```
```bash
  npm i
```
```bash
  npm run dev
```
